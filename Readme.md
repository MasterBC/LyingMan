# LyingMan
一款在线狼人杀游戏平台
### 运行
- 安装[NodeJS v4.6.0](https://nodejs.org/en/)
- 切换路径 `cd <LyingMan>\server`
- 执行 `node .\server.js`
- 浏览器浏览 `localhost:81`
